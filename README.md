## Dependencies:

### Config file

To provide authenication credentials -> `~/.tower_cli.cfg`:

```bash
host: https://<your_instance>
verify_ssl: True
username: <your_user>
password: <your_pass>

```

### Python libraries

```bash
pip install ansible-tower-cli
```

### Collections

```bash
ansible-galaxy collection install -r collections/requirements.yml
```
